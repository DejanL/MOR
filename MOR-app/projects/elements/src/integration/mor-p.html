MOR-P (integration demo)

The integration of MOR-P component is possible via Web Components, where custom element de4a-mor-p is available.

The following minimal example depicts that developer needs to include the following files (available in project's Github):

  • CSS file mor-elements.css,
  • JS file mor-elements.js
and custom element de4a-mor-p with the following attributes:
  • default-lang,
    • single value from → en, sl, es, pt, fr,
  • post-action-value,
    • multiple (comma separated values) from → BirthCertificate, MarriageCertificate,
confirmedCanonicalEvidenceTypes is the name of output variable, stored in Local Storage.
