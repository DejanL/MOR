MOR-ER (integration demo with Spanish portal CSS)

The integration of MOR-ER component is possible via Web Components, where custom element de4a-mor-er is available.

The following minimal example depicts that developer needs to include the following files (available in project's Github):

  • CSS file mor-elements.css and optional additional CSS rules that override the current layout (see example),
  • JS file mor-elements.js
and custom element de4a-mor-er with the following attributes:
  • default-lang,
    • single value from → en, sl, es, pt, fr,
  • requester-country-code,
    • single value from → AT, BE, BG, HR, CY, CZ, DK, EE, FI, FR, DE, EL, HU, IE, IT, LV, LT, LU, MT, NL, PL, PT, RO, SK, SI, ES, SE,
  • canonical-evidence-types,
    • multiple (comma separated values) from → BirthCertificate, MarriageCertificate,
  • output-j-s-array-id,
    • name of output variable, stored in Local Storage (e.g. outputJSArrayIdMorEr)

Source code (example with Spanish portal CSS)

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <title>MOR ER</title>
    <link rel="stylesheet" href="mor-elements.css" />
    <style>
      de4a-mor-er {
        color: #170f40 !important;
      }
      de4a-mor-er code.text-primary {
        color: #fcc415 !important;
      }
      de4a-mor-er button.btn-secondary,
      de4a-mor-er div.modal-footer button.btn-light {
        background: #3d9be9 !important;
        border: 1px solid #3d9be9 !important;
        color: #fff !important;
      }
      de4a-mor-er button.btn-secondary:hover,
      de4a-mor-er div.modal-footer button.btn-light:hover {
        background: #187fd5 !important;
        border-color: #187fd5 !important;
      }
    </style>
    <script src="mor-elements.js"></script>
  </head>
  <body>
    <div class="de4a-bs">
      <de4a-mor-er
        default-lang="en"
        requester-country-code="BE"
        canonical-evidence-types="BirthCertificate,MarriageCertificate"
        output-j-s-array-id="outputJSArrayIdMorEr"
      ></de4a-mor-er>
    </div>
  </body>
</html>

Building from source (technical details)

Source code of Angular application is availale in MOR-app folder and MOR-ER Web Component can be regenerated with the following command npm run build:elements.
